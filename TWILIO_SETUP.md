# Configuración Twilio - Pasos

## PARA PRUEBAS (GRATIS)
1. Ve a: https://console.twilio.com
2. Phone Numbers → Manage → Verified Caller IDs
3. Agregar tu número: +591XXXXXXXX
4. Verificar con el código que te envíen
5. ¡Listo para probar!

## PARA PRODUCCIÓN (CUALQUIER NÚMERO)
1. Ve a: https://console.twilio.com/billing
2. Actualizar cuenta a plan de pago ($20 USD mínimo)
3. ¡Ya funciona con cualquier número!

## NO CAMBIAR CÓDIGO
- El código ya está listo para ambos casos
- Solo cambia la configuración en Twilio, no el código
